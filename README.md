##  Overview

**SafeDrive** is a full-stack web application focused on improving road safety through centralized monitoring, intuitive dashboards, and structured driver-assistance features.  
It integrates a responsive frontend interface with backend services to provide scalable and organized system functionality.

---

##  Features

- Driver Safety Dashboard  
- Monitoring & Visualization Interface  
- REST API Integration  
- Real-time User Interaction  
- Modular Frontend Structure  
- Clean Backend Architecture  
- Responsive Web Layout

##  Getting Started

### 1️ Clone the Repository

```bash
git clone https://github.com/jordannealexie/safedrive-1.0.git
cd SafeDrive

2️⃣ Run Backend Server
cd SafeDrive_Backend
npm install
npm start

Server runs at:

http://localhost:3000
3️⃣ Launch Frontend

Open:

SafeDrive_Frontend/index.html

Or run using VS Code Live Server.

📂 Folder Structure
SafeDrive/
│
├── SafeDrive_Backend/
│   ├── routes/
│   ├── controllers/
│   ├── config/
│   └── server.js
│
├── SafeDrive_Frontend/
│   ├── css/
│   ├── js/
│   ├── assets/
│   └── index.html
```
